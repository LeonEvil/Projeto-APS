Propósito de Desenvolvimento

Sobre o Projeto

O projeto movimento sustentável tem como objetivo, conscientiza as pessoas sobre a importância da ecologia/reciclagem no mundo. Pois grande parte da população não sabe como fazer corretamente uma ação sustentável. E para isso o nosso projeto adquiri-o o título de movimento sustentável, para mostra de forma pratica/rápida as pessoas o que pode ser feito para conscientizar ou ensinar todas, de uma forma positiva onde não haja resultados negativos para uma geração futura.

Desenvolvimento

A equipe responsável por este site, é um grupo de seis (6) integrantes, cuja conhe-cimento acadêmico dos mesmos é em diversas áreas da computação. Este site foi criado no intuito de fornecer informações ecológicas para o mundo, e para a aprovação acadêmi-ca dos integrantes em um curso superior. Entretanto as ferramentas utilizadas para criação do mesmo, foram a codificação de HTML que é totalmente responsável para estrutura do site dando o fornecimento base. Tendo também a ajuda do CSS que auxilio o desenvolvimento de design (cor, fonte e ou-tros). e logo após o JAVASCRIPT que é totalmente responsável pela funcionalidade do site. Sendo assim, tivermos um resultado positivo na criação do site, não obtendo nem uma falha no sistema ou erro em sua funcionalidade. A IDE utilizada para o desenvolvimento so site foi Visual Studio e Sublime Text, dando total auxilio para receber cada linha de condificação. Sendo assim tornando um site de fácil acesso, simples e sem complicações para entender o conteúdo do mesmo. Dando-se então um bom site para o usuário utulizar.
